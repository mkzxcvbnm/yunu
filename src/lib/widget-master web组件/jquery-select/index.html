$(".select1").select();

调用方法：

$(selector).select(options,callback(api));

options参数

参数 默认值 说明
selectCls 'select' 选择框外层class
boxCls 'box' 选择框class
innerCls 'inner' 下拉菜单外层class
listCls 'list' 下拉菜单class
activeCls 'active' 下拉菜单选中class
html(status) function(status){ status.list.append("<li>"+status.item.text()+"</li>"); } 下拉菜单自定义结构
onSelect(value) [无] 选择事件处理

callback(api)参数

方法 说明
setValue() 修改当前值
